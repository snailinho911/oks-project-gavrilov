Комментарий для бабушки: Это наша карта сети. Круглое устройство — роутер (главный), прямоугольное — коммутатор (распределитель), а компьютеры — это конечные пользователи.

2. Ключевые этапы настройки

2.1. Настройка имени роутера

Первым делом, мы дали роутеру уникальное имя — мою фамилию. Это важно для идентификации устройства в сети.

Комментарий для бабушки: Мы как бы повесили табличку с фамилией на дверь главного устройства, чтобы не перепутать его с другими.

2.2. Настройка IP-адреса на роутере

Мы назначили роутеру статический IP-адрес 192.168.25.1. Третий октет (25) — мой номер по списку. Этот адрес будет служить шлюзом для всей нашей локальной сети.

Комментарий для бабушки: Мы присвоили главному устройству уникальный номер дома (IP-адрес), чтобы другие устройства знали, куда отправлять письма (данные) за пределы их улицы (локальной сети).

2.3. Настройка IP-адресов на ПК

Каждому компьютеру был присвоен свой уникальный адрес в той же сети (192.168.25.10 и 192.168.25.11) и указан адрес шлюза.

Комментарий для бабушки: Каждому компьютеру мы дали свой номер квартиры, чтобы они могли общаться между собой и знали, через какую дверь (шлюз) выходить на улицу.
